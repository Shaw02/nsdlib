NSDL（NES Sound Driver Library）

使い方（Usage）

サウンドドライバー（nsd.lib）

サウンドドライバー本体です。
オブジェクトライブラリとして開発しています。

This is sound driver.
It was developed as object library.

ＭＭＬコンパイラ（nsc.exe）

開発中！
MMLコンパイラーです。
MMLで記述されたBGM, SEを、本サウンドドライバーで再生可能なデータに変換します。

This is MML compiler.
It can convert the MML(Music Macro Langage) to the deta for this sound driver.
