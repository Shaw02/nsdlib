内容（Contents）

現在のトラックの調号を指定します。

注意（Notes）

ＭＭＬコンパイラーの疑似命令です。
次にK{}コマンドが来るまで、本コマンドで指定された音階に、自動で臨時記号が付加されてコンパイルされます。

各記述ブロック（Sub, BGM, SE）では、デフォルトは調号無しとなります。
従い、サブルーチンコールする場合は、サブルーチン先も調号を指定して下さい。

例（Sample）

    K{#fc}   defgab>cd    /* D-Dur scale */
    K{=fc}   cdefgab>c    /* C-Dur scale */
