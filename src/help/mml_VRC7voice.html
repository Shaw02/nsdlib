NSDL（NES Sound Driver Library）

MML reference: VRC7 User Instrument

定義（Define）

        @Vn

引数（Inputs）

n

ユーザー定義音色（@コマンドの0番の音色）に設定する、VRC7記述ブロックの番号を指定します。

内容（Contents）

VRC7音源の、ユーザー定義音色（@コマンドの0番の音色）をVRC7記述ブロックn番の音色に設定します。

注意（Notes）

例（Sample）


VRC7(1){	//Piano
	//	TL FB
	@	18  4
	//	AR DR SL RR KL MT AM VB EG KR DT
		15  1  3  3  2  1  0  0  0  1  1
		15  2  3  1  0  1  0  0  0  1  0
}

bgm(0){
	TR6	l4	o4	v15	Rm2	@0	@V1	t120	cdefgab> c1
}
